# 滚轮选中高亮形状调整设计

## 目标

修复日期时间滚轮中间选中高亮为圆形时，较宽文字（例如“今天”）视觉上超出圆形框的问题。将该高亮改为带轻微圆角的方形，同时保持滚轮列、文字、颜色、尺寸和交互逻辑不变。

## 方案

调整 `src/app/globals.css` 中 `.up-wheel-highlight` 的 `border-radius`，从完全圆形改为小圆角（目标值 `12px`）。由于高亮层已覆盖滚轮中间选中区域，组件结构和滚动逻辑无需修改。

暗色主题继续复用同一高亮层，仅保留现有暗色背景和阴影覆盖，不新增主题分支。

## 验证标准

- 选中高亮呈横向方形并带轻微圆角，而非胶囊/圆形。
- “今天”等较宽选中文字完全位于高亮区域内，不被圆弧边缘造成溢出感。
- 滚轮滚动、点击选择、日期时间值和明暗主题行为不变。
- `npm run lint` 和 `npm run build` 通过。
