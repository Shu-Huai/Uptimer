# 日期时间选择器指针拖拽设计

## 目标

为添加记录表单中的日期和时间选择器增加平滑的鼠标拖拽交互：用户按住日期滚轮横向拖动，或按住小时、分钟、秒滚轮纵向拖动，滚轮内容跟随指针移动；松开后自动平滑吸附到最近的选项。

## 现状

`DateTimeWheelPicker` 通过复用的 `WheelColumn` 渲染日期、小时、分钟和秒列。组件目前支持点击选项、原生滚动和横向滚轮事件，并在滚动停止后根据固定项目尺寸计算最近选项。日期列使用横向滚动，时间列使用纵向滚动。

## 方案

在 `WheelColumn` 内使用 Pointer Events 统一处理鼠标和触摸拖拽：

- `pointerdown` 记录指针起点和滚动起点，捕获当前指针，并标记拖拽状态。
- `pointermove` 根据指针位移直接更新对应的 `scrollLeft` 或 `scrollTop`，保证拖动过程跟手。
- 拖动期间暂停 scroll snap，避免浏览器在每次移动时抢先吸附。
- `pointerup`、`pointercancel` 和丢失捕获时结束拖拽，按当前滚动位置计算最近选项，并通过平滑滚动吸附。
- 仅发生短距离点击时保留现有选项按钮的点击行为；实际拖动不触发选项点击。
- 现有受控值同步、原生滚动结束处理和横向滚轮处理继续复用，不改变日期/时间值的格式或边界限制。

通过一个拖拽状态 class 和 `touch-action` 样式区分拖拽过程，避免引入第三方依赖。拖拽时不修改业务状态，只有吸附确认后才调用 `onSelect`，减少连续拖动导致的表单更新。

## 验证

- 添加针对拖拽位置计算/吸附行为的单元测试，覆盖横向和纵向项目尺寸以及边界夹取。
- 运行现有测试和 lint。
- 手动验证日期、小时、分钟、秒四个滚轮的按住拖动、松开吸附、点击选择、触摸滚动和鼠标滚轮行为。
